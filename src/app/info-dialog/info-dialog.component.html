Files format info

Attributes file

The attributes file should be a json file that contains a list of attribute objects that must have the following fields:
- name(the name of the attribute. ex: qualification)
- description(optional description of the attribute. ex: The qualification needed for the job)
- type(the type of the field, so what you want to get back from the llm. ex: string will return a string or list[string] will return a list of strings)

Attributes values file

The attributes values file should be a json file that contains a list of attribute value objects that must have the following fields:
- name(the name of the attribute value. ex: Full-time)
- description(optional description of the attribute value. ex: Work 8 hours a day, 40 hours a week.)
- attribute(the name of the attribute that this value corresponds to. ex: qualification)

Memory file

The memory file should be a json file that contains a list of json objects that must have the following fields:
- memory(this is a json object that has the next fields)
- field_of_activity(the domain of the classification)
- input(The input for the llm.)
- output(The output expected from the llm.)
